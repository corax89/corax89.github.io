Blockly Blocks Reference

Basic Object Parameters

Every object in Blockly has a set of parameters that define its behavior and appearance. Below is a complete list of all available parameters with detailed descriptions:

Parameter Code Type Description
X x Number Horizontal coordinate of the object (in pixels). Measured from the left edge of the screen.
Y y Number Vertical coordinate of the object (in pixels). Measured from the top edge of the screen.
Width width Number Object width in pixels. Determines horizontal collision size.
Height height Number Object height in pixels. Determines vertical collision size.
Speed X speedx Number Horizontal movement speed (pixels per frame). Positive values - right, negative - left.
Speed Y speedy Number Vertical movement speed (pixels per frame). Positive values - down, negative - up.
Name name String Contains the name of the object, if the object is created from a prototype then it will have the prototype name
Visible visible Boolean Determines whether the object is visible on screen. When false, the object isn't rendered but still participates in physics.
Solid solid Boolean If true, the object will collide with other solid objects. If false - the object will be passable.
Angle angle Number Rotation angle in degrees (0-360). Rotation occurs around the object's center.
Flip flip Boolean If true, the object's sprite will be horizontally flipped (mirrored).
Mass mass Number Object mass for physics calculations. Default is 1.0. Higher mass makes the object harder to move.
Restitution restitution Number (0-1) Bounciness coefficient during collisions. 0 - no bounce, 1 - perfectly elastic bounce.
Is Static isStatic Boolean If true, the object won't move under physics (like a platform or wall).
Z-index zIndex Number Determines rendering order. Objects with higher zIndex are drawn above objects with lower values.
Is On Ground? isOnGround Boolean Automatically set to true if the object is standing on another solid object. Read-only.

Game Loop

Infinite Game Loop
Main game loop that runs every frame. You can enable/disable screen clearing.

Controls

Is Button Held?
Checks if the specified button (arrows or A/B/X/Y) is being held.
Is Button Pressed?
Checks if the specified button was just pressed.
Get Joystick Axis Value
Returns joystick axis value (-1.0 to 1.0).

Drawing

Draw Point
Draws a point at specified coordinates with given color.
Draw Text
Displays text at specified coordinates with given size and color.
Draw Sprite
Draws a sprite (image) at specified coordinates with given size.
Clear Screen
Fills the entire screen with specified color.
Load Sprite
Allows you to upload or draw a picture.
Color Picker
Allows selecting a color from palette.

Objects

Create a prototype of an object
The object prototype is used to create objects and in the level editor.
Create a new object from a prototype
Creates an object with parameters from the prototype at point X Y
Create New Object
Creates a new game object with specified parameters (position, size, sprite).
Clone Object
Creates a copy of existing object.
Set Object Parameter
Changes value of object parameter (X, Y, width, height etc.).
Get Object Parameter
Returns value of object parameter.
Add to Object Parameter
Increases object parameter value by specified number.
Do Every Frame
Sets code that will execute for the object every frame.
Do on Collision
Sets code that will execute when object collides with another.
Control Object
Allows controlling object with keys or joystick.
Move to Point
Makes object move to specified point with given speed.
Iterate All Objects
Executes code for each existing object.

Logic

Check Rectangle Collision
Checks if two rectangles intersect.
If Outside Screen
Checks if object went outside screen boundaries.

Sound

Melody Editor
Allows creating melodies in visual editor.
Play Melody
Plays melody with specified tempo.

System

Execute After
Executes code after specified time (in milliseconds).
Get Time
Returns current time in milliseconds.
Set Gravity
Sets global gravity for all objects.
Is Touching?
Checks if user is touching the screen.
Get Touch
Returns touch screen coordinates.

Variables

Create Local Variable
Creates new variable with specified name and value.
Get Local Variable
Returns value of local variable.

Advanced

JS Code
Allows inserting arbitrary JavaScript code.

Logical Operators
Blocks for working with boolean values: AND, OR, NOT, comparisons (==, !=, <, >, <=, >=).
Conditionals
Blocks for branching logic: if, if-else.
Loops
Blocks for creating loops: while, for, repeat.
Math Operations
Blocks for mathematical calculations: +, -, *, /, %, exponent, root, trigonometric functions.
Text
Blocks for working with strings: concatenation, length, substring, contains check.
Lists
Blocks for working with arrays: create list, access by index, add/remove elements, list length.
Functions
Blocks for creating and calling custom functions with parameters and return values.
Variables
Blocks for creating variables, assigning values and getting variable values.
Comments
