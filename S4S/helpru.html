Основные параметры объектов

Каждый объект в Blockly имеет набор параметров, которые определяют его поведение и внешний вид. Ниже приведен полный список всех доступных параметров с подробным описанием:

Параметр Код Тип Описание
X x Число Горизонтальная координата объекта (в пикселях). Отсчитывается от левого края экрана.
Y y Число Вертикальная координата объекта (в пикселях). Отсчитывается от верхнего края экрана.
Ширина width Число Ширина объекта в пикселях. Определяет горизонтальный размер коллизии.
Высота height Число Высота объекта в пикселях. Определяет вертикальный размер коллизии.
Скорость X speedx Число Горизонтальная скорость движения объекта (пикселей за кадр). Положительные значения - вправо, отрицательные - влево.
Скорость Y speedy Число Вертикальная скорость движения объекта (пикселей за кадр). Положительные значения - вниз, отрицательные - вверх.
Видимость visible Логический Определяет, виден ли объект на экране. При false объект не отрисовывается, но продолжает участвовать в физике.
Твердость solid Логический Если true, объект будет сталкиваться с другими твердыми объектами. Если false - объект будет проходимым.
Угол angle Число Угол поворота объекта в градусах (0-360). Поворот происходит вокруг центра объекта.
Отражение flip Логический Если true, спрайт объекта будет отражен по горизонтали (зеркально).
Масса mass Число Масса объекта для физических расчетов. По умолчанию 1.0. Чем больше масса, тем сложнее сдвинуть объект.
Упругость restitution Число (0-1) Коэффициент упругости при столкновениях. 0 - объект не отскакивает, 1 - идеально упругий отскок.
Статичность isStatic Логический Если true, объект не будет двигаться под действием физики (как платформа или стена).
Глубина прорисовки zIndex Число Определяет порядок отрисовки объектов. Объекты с большим zIndex рисуются поверх объектов с меньшим значением.
Стоит на поверхности? isOnGround Логический Автоматически устанавливается в true, если объект стоит на другом твердом объекте. Только для чтения.

Игровой цикл

Бесконечный игровой цикл
Основной цикл игры, который выполняется каждый кадр. Можно включить/отключить очистку экрана.

Управление

Кнопка удерживается?
Проверяет, удерживается ли указанная кнопка (стрелки или A/B/X/Y).
Кнопка нажата?
Проверяет, была ли только что нажата указанная кнопка.
Получить значение оси стика
Возвращает значение оси джойстика (-1.0 до 1.0).

Рисование

Нарисовать точку
Рисует точку в указанных координатах заданным цветом.
Нарисовать текст
Выводит текст в указанных координатах с заданным размером и цветом.
Нарисовать спрайт
Рисует спрайт (изображение) в указанных координатах с заданным размером.
Очистить экран
Заполняет весь экран указанным цветом.
Загрузка спрайта
Позволяет загрузить спрайт в формате png. Возвращает номер загруженного спрайта, каждый вызов будет заргужать спрайт заново и занимать место в памяти, поэтому лучше загружать спрайт в начале игры и сохранять в переменной.
Выбор цвета
Позволяет выбрать цвет из палитры.

Объекты

Создать новый объект
Создает новый игровой объект с указанными параметрами (позиция, размер, спрайт).
Клонирование объекта
Создает копию существующего объекта.
Изменить параметр объекта
Изменяет значение параметра объекта (X, Y, ширина, высота и др.).
Получить параметр объекта
Возвращает значение параметра объекта.
Добавить к параметру объекта
Увеличивает значение параметра объекта на указанное число.
Делать каждый кадр
Устанавливает код, который будет выполняться для объекта каждый кадр.
Делать при столкновении
Устанавливает код, который будет выполняться при столкновении объекта с другим.
Управление объектом
Позволяет управлять объектом с помощью клавиш или джойстика.
Идти к точке
Заставляет объект двигаться к указанной точке с заданной скоростью.
Перебор всех объектов
Выполняет код для каждого существующего объекта.

Логика

Проверка столкновения квадратов
Проверяет, пересекаются ли два прямоугольника.
Если выход за экран
Проверяет, вышел ли объект за границы экрана.

Звук

Редактор мелодии
Позволяет создать мелодию в визуальном редакторе.
Проиграть мелодию
Воспроизводит мелодию с указанным темпом.

Системные

Выполнить через
Выполняет код через указанное время (в миллисекундах).
Получить время
Возвращает текущее время в миллисекундах.
Установить скорость падения
Устанавливает глобальную гравитацию для всех объектов.
Есть ли касание?
Проверяет, касается ли пользователь экрана.
Получить касание
Возвращает координаты касания экрана.

Переменные

Создать локальную переменную
Создает новую переменную с указанным именем и значением.
Получить локальную переменную
Возвращает значение локальной переменной.

Дополнительно

JS код
Позволяет вставить произвольный JavaScript код.

Стандартные блоки Blockly

Логические операторы
Блоки для работы с логическими значениями: И, ИЛИ, НЕ, сравнения (==, !=, <, >, <=, >=).
Условия
Блоки if, if-else для ветвления логики программы.
Циклы
Блоки для организации циклов: while, for, repeat.
Математические операции
Блоки для математических вычислений: +, -, *, /, %, возведение в степень, корень, тригонометрические функции.
Текст
Блоки для работы со строками: конкатенация, длина строки, подстрока, проверка наличия подстроки.
Списки
Блоки для работы с массивами: создание списка, доступ по индексу, добавление/удаление элементов, длина списка.
Функции
Блоки для создания и вызова пользовательских функций с параметрами и возвращаемыми значениями.
Переменные
Блоки для создания переменных, присваивания значений и получения значений переменных.
Комментарии
Блоки для добавления поясняющих комментариев к коду.
